# 图书馆管理系统 python1136

## 项目概述

图书馆管理系统是一个基于Python语言的Web应用，结合了Django框架和Vue.js技术，实现了对图书馆的日常业务进行管理的功能。

## 技术栈

- **后端：**
  - Python 3.7
  - Django
  - MySQL
- **前端：**
  - Vue.js
- **开发环境：**
  - PyCharm

## 功能模块

系统主要包括以下功能模块：

- **首页**
- **个人中心**
- 用户管理
- 图书馆管理
- 图书信息管理
- 图书类型管理
- 借阅信息管理
- 归还信息管理
- 续借信息管理
- 罚金信息管理
- 留言板管理
- 系统管理

## 系统角色

系统主要服务于图书馆管理员和普通用户，实现图书馆资源的有效管理。

## 运行环境

- Python 3.7及以上版本
- MySQL数据库
- 适用于主流Web服务器环境
